# OOP Semester 5


## Лабораторні роботи

### Lab Work 1 (V-6) - [Gemstones Management System](./lab1/)

Консольна програма для управління колекцією дорогоцінних та напівдорогоцінних каменів. Система дозволяє створювати намисто з каменів, додавати нові камені, сортувати їх за різними критеріями, шукати за прозорістю та зберігати/завантажувати дані з файлу.

### Lab Work 2 (V-7) - [XML Parsing and Processing](./lab2/)

Проект демонструє роботу з XML файлами для обробки даних про холодну зброю (ножи). Реалізує три різні підходи до парсингу XML (SAX, DOM, StAX), валідацію XML документів за XSD схемою, сортування колекцій за допомогою Comparator, та трансформацію XML даних в HTML формат.

### Lab Work 3 (V-4) - [Maze Game (Android)](./lab3/)

Android додаток-гра, де гравець має знайти вихід з випадково згенерованого лабіринту. Реалізовано в Android Studio.
