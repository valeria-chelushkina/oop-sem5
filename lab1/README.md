# Gemstones Management System

## Опис проекту

Цей проект є системою управління намистом з дорогоцінних та напівдорогоцінних каменів. Програма дозволяє створювати намисто, додавати до нього різні типи каменів, сортувати їх, шукати за різними критеріями та зберігати/завантажувати дані з файлів.

## Основні можливості

- **Додавання каменів**: Підтримка різних типів дорогоцінних та напівдорогоцінних каменів
- **Сортування**: Сортування каменів за вартістю, вагою або цінністю (вартість/карат)
- **Пошук**: Пошук каменів за діапазоном прозорості
- **Збереження/Завантаження**: Робота з файлами для збереження та завантаження намиста
- **Створення власних каменів**: Можливість створювати камені з власними назвами

## Класи каменів

### Дорогоцінні камені (PreciousStone)
- **Diamond** - Діамант (має параметр cut - огранка)
- **Ruby** - Рубін (має параметр clarity - чистота)
- **Emerald** - Смарагд (має параметр treated - оброблений/необроблений)

### Напівдорогоцінні камені (SemiPreciousStone)
- **Amethyst** - Аметист (має параметр pattern - візерунок)
- **Opal** - Опал (має параметр playOfColor - гра кольорів)
- **Aventurine** - Авентурін (має параметр inclusions - включення)

### Власні камені
- **PreciousStone** - можна створити власний дорогоцінний камінь
- **SemiPreciousStone** - можна створити власний напівдорогоцінний камінь


## Меню програми

1. **Show the necklace** - Показати інформацію про намисто
2. **Add gemstone** - Додати камінь до намиста
   - Можна вибрати зі списку або створити власний
3. **Sort gemstones** - Сортувати камені
   - За цінністю (cost/carats)
   - За ціною
   - За вагою
4. **Find gemstone by transparency** - Знайти камені за прозорістю
   - Автоматично перевіряє та виправляє діапазон [0-1]
5. **Save the necklace** - Зберегти намисто у файл
   - Можна вибрати дефолтний файл або вказати власний шлях
6. **Load the necklace** - Завантажити намисто з файлу
   - Можна вибрати дефолтний файл або вказати власний шлях
0. **Exit** - Вийти з програми

## Формат файлу

Намисто зберігається у текстовому файлі з роздільником `|`:

```
ClassName|Name|Weight|Cost|Transparency|Lustre|Color|AdditionalField1|AdditionalField2
```

Приклад:
```
Diamond|Diamond|2.5|50000.0|0.95|Adamantite|White|Africa|Round brilliant
Ruby|Ruby|1.8|30000.0|0.85|Subadamantine|Red|Myanmar|0.9
```
